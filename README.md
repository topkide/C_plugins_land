# Land (부동산) 플러그인

1.21.4 Paper 단일 서버용 토지 구매·보호·판매 플러그인입니다.
기존 Skript 화폐(currency.sk)·상점(shop.sk)을 **수정하지 않고** 위에 얹는 방식으로 동작합니다.

---

## 빌드

Core 플러그인과 동일한 툴체인(Gradle 8.8 wrapper · shadow 8.3.9 · Java 21)입니다.

```bash
./gradlew shadowJar
```

산출물: `build/libs/Land-1.0.0-all.jar`

> 이 프로젝트는 Paper 저장소(repo.papermc.io)에서 `paper-api:1.21.4` 를 받아 빌드합니다.
> 인터넷이 되는 개발 환경에서 위 명령을 실행하면 됩니다.

---

## 설치

1. `build/libs/Land-1.0.0-all.jar` 를 서버 `plugins/` 에 넣고 서버를 재시작합니다.
2. 환불을 즉시 지급 방식(BRIDGE, 기본값)으로 쓸 경우, `bridge/land_bridge.sk` 를
   `plugins/Skript/scripts/` 에 넣고 `/sk reload land_bridge` 를 실행합니다.
   (환불을 수표(VOUCHER) 방식으로 바꾸면 이 파일은 필요 없습니다 — config 참고)

---

## 사용법

### 운영자 (`/부동산관리`, 권한 `land.admin`)

1. `아이템지정` — 손에 든 아이템(예: ItemsAdder 아이템)을 구매서 기본 디자인으로 지정
2. `만들기 <가로> <세로> <가격> <화폐> [이름]` — 토지권 생성
   예) `/부동산관리 만들기 30 30 50000 골드`
3. `지급 <플레이어> <이름> [개수]` — 구매서 지급
4. 기타: `삭제 <이름>`, `목록`, `정보 [플레이어]`, `reload`

> 크기별로 로어에 크기가 표시되어 자동으로 구분됩니다. 같은 디자인으로 서로 다른 크기를
> 여러 개 만들 수 있습니다.

### 플레이어

- 구매서 **우클릭** → 선택 모드 시작 (바라보는 곳에 범위 미리보기)
- 원하는 위치를 바라보고 **다시 우클릭** → 구매 (빨간색이면 이미 팔린 땅)
- **웅크리고 우클릭** → 선택 취소
- 인접한 내 토지는 구매 시 **자동으로 하나로 합쳐집니다**
- `/부동산` → 내 토지 GUI (정보 / 보호 ON·OFF / 텔레포트 / 판매)
- `/부동산 보호` → 지금 서 있는 내 토지의 보호를 토글

## 상점 연동 (되팔기 방지)

구매서는 상점 스크립트를 **수정하지 않고** 그대로 상점에 등록/판매할 수 있습니다.
(상점 저장은 재질·모델·이름·로어를 보존하므로 우클릭 시 정확히 식별됩니다.)

되팔기(계약서를 상점에 되파는 것)를 막으려면, **상점에 구매서를 등록할 때 판매가격(sell-price)을
설정하지 마세요.** shop.sk 는 판매가격이 0이면 판매를 막습니다. 별도 스크립트 수정이 필요 없습니다.

## 위임받아 정한 기본값 (모두 config.yml 에서 변경 가능)

| 항목 | 기본값 |
|---|---|
| 격자 스냅 | 켬 / 단위 10 |
| 환불 비율 | 80% |
| 환불 방식 | BRIDGE(즉시 입금) — VOUCHER(수표)/NONE 선택 가능 |
| 판매 단위 | 부지 통째 |
| 보호 높이 | Y 전체 |
| 선택 취소 | 웅크리고 우클릭 |
| 1인 최대 소유 | 무제한 |
| 되팔기 안전망 | 문서 안내(상점 무수정) |

## 데이터

`plugins/Land/data/` 아래에 `lands.yml`, `certificates.yml` 로 저장됩니다.
자동 저장 주기는 config 의 `misc.autosave-minutes` (기본 5분), 종료 시에도 저장합니다.

저장소는 인터페이스(`storage/Storage`)로 분리되어 있어, 추후 멀티서버(MySQL/Redis)로
확장할 때 이 인터페이스만 새로 구현하면 됩니다.
